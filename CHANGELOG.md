# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-06-03

### Added
- Initial release with 5 ad organisms: Banner, Interstitial, Reward, Native, Inline
- Atomic design architecture (Atoms, Molecules, Organisms)
- Media engine with expo-image and expo-av support
- useAdLogic, useAdTimer, useAdVisibility, useAdMedia hooks
- ThemeProvider with full dark mode support
- Animation system (fade, slide, scale enter/exit)
- Gesture handler integration for drag interactions
- Accessibility support (screen readers, roles, hints)
- App Store / Play Store compliance features
- Comprehensive test suite with Jest
- Example Expo app with all ad types
- CI/CD with GitHub Actions
- TypeScript strict mode throughout
